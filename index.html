About Visual Studio Code

And the Console

Visual Studio Code is a coding IDE platform that is for advanced coders that wish to bump up their coding IDE. VS Code, as it is abbreviated, has many more features than other coding IDEs, such as Brackets. You can do many mathematical operations in VS Code, whether it is finding the maximum number in a set, or taking the power of 2 numbers. There are also functions for strings in VS Code. All you have to do is go to code.visualstudio.com, and click the download button for your OPS, whether it is Windows, Macintosh, or Linux.

To open the console, right click on a page, and click on the Inspect button. Or, if you want to use the shortcut, just do Cntrl + Shift + I, and the inspect page will open. After that, all you need to click on is the console button, and you will see the outputs and errors (if you have any) of your code. You can use console.log in your JavaScript code in order to see your results in the console. Also, you can see precisely which line you have the errors in your code using the console. Just look for the red boxes, and look at the right hand side of them. you will see the file name, and what line of code the error is in.
